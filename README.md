# TintoraAI

TintoraAI — автономный инструмент для раскраски чёрно-белых фотографий, созданный [T1desLuck](https://github.com/T1desLuck). Проект использует U-Net с perceptual loss, поддерживает любые размеры изображений и обучается с нуля. Планируется добавить GAN для улучшения качества и постобработку для реализма.

- **[Установка](INSTALL.md)**: Как настроить проект.
- **[Использование](USAGE.md)**: Как раскрашивать изображения.
- **[Обучение](TRAINING.md)**: Инструкции по обучению модели.
- **[Контрибьютинг](CONTRIBUTING.md)**: Как внести вклад.

## О проекте
TintoraAI обрабатывает старые фото, исправляя пятна и искажения, понимает контекст (человек, одежда, фон) и избегает размытия. Планируется разделить GAN для улучшения качества как отдельную модель с автоматической последовательностью.

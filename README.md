# TintoraAI — Раскраска черно-белых фотографий

Добро пожаловать в **TintoraAI**! Это нейросеть для автоматической раскраски черно-белых фотографий с использованием глубокого обучения. Она проста в использовании, полностью автономна и поддерживает разные стили. Даже если ты новичок, этот проект для тебя!

## Что такое TintoraAI?
TintoraAI — это программа, которая:
- Превращает черно-белые фотографии в цветные с помощью U-Net и классификатора объектов.
- Учитывает семантику изображения (распознает объекты, например, глаза или дерево) с помощью 100 классов от `NpyLabelNet`.
- Поддерживает стили: `modern` (яркие цвета), `vintage` (приглушенные тона), `neutral` (стандартные цвета).
- Работает локально, без внешних сервисов или данных.
- Может обучаться на твоих собственных фотографиях (512x512).

## Возможности
- **Инференс**: Раскрашивай одно фото за раз с настройкой стиля и насыщенности.
- **Обучение**: Настрой модель на своем датасете (черно-белые и цветные фото + `.npy` метки).
- **Стилизация**: Выбирай между современным, винтажным или нейтральным стилем.
- **Метрики качества**: SSIM, LPIPS и точность классификации для оценки результатов.
- **Gradient Accumulation**: Поддержка обучения на больших батчах даже на слабом оборудовании.
- **Автономность**: Все веса и данные хранятся локально, никаких облачных API.

## Как подготовить датасет
Для обучения нужен датасет с тремя папками:
- `bw/`: Черно-белые изображения (`.jpg` или `.png`, 512x512 пикселей).
- `color/`: Цветные версии тех же изображений (те же имена файлов, 512x512).
- `labels/`: Метки объектов в формате `.npy` (числа от 0 до 99, созданные `NpyLabelNet`).
Подробности и примеры — в [TRAINING.md](TRAINING.md).

# TintoraAI

TintoraAI — это нейросеть для раскраски старых, выцветших и черно-белых фотографий, превосходящая по качеству и стилю такие решения, как DeOldify и Photoshop. Она обеспечивает фотореалистичные цвета, работает с любыми размерами изображений и поддерживает локальный/облачный запуск на CPU/GPU.

## Особенности
- Уникальная архитектура с attention-механизмами для детальной раскраски.
- Простой веб-интерфейс (Streamlit) и API (FastAPI).
- Конфигуратор для настройки насыщенности и стиля.
- Поддержка любых размеров изображений без потери качества.
- Обучение с нуля для уникального стиля.
- Фотореализм с учетом пиксельных деталей.

# TintoraAI — Раскраска чёрно-белых фото

Добро пожаловать в **TintoraAI**! Это программа, которая раскрашивает старые чёрно-белые фотографии с помощью искусственного интеллекта. Даже если ты новичок, ты сможешь её установить и использовать!

## Что это такое?
TintoraAI — это нейросеть, которая:
- Превращает чёрно-белые фото в цветные.
- Поддерживает разные стили (современный, винтажный, нейтральный).
- Умеет распознавать объекты на фото.

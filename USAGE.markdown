# Использование TintoraAI

Этот документ описывает, как использовать TintoraAI через веб-интерфейс (Streamlit) и API (FastAPI).

## Веб-интерфейс (Streamlit)
1. **Запустите интерфейс**:
   ```bash
   streamlit run src/ui/app.py
   ```
   Откройте браузер по адресу `http://localhost:8501` (или ссылку от ngrok в Colab).

2. **Использование**:
   - **Загрузка фото**: Нажмите "Выберите фото" и загрузите черно-белое или выцветшее изображение (jpg, jpeg, png).
   - **Настройки**:
     - В боковой панели выберите **Насыщенность** (0.5–2.0) для контроля яркости цветов.
     - Выберите **Стиль** ("Современный", "Винтаж", "Нейтральный") для желаемого эффекта.
   - Нажмите **Раскрасить** для обработки.
   - Скачайте результат, нажав **Скачать раскрашенное фото**.

3. **Особенности**:
   - Поддерживает любые размеры изображений без потери качества.
   - Не изменяет оригинал, создает новую версию.
   - Учитывает мелкие детали благодаря attention-механизмам.

## API (FastAPI)
1. **Запустите API**:
   ```bash
   uvicorn src.api.app:app --host 0.0.0.0 --port 8000
   ```

2. **Пример запроса**:
   ```bash
   curl -X POST -F "file=@image.jpg" -F "saturation=1.2" http://localhost:8000/colorize
   ```
   - `file`: Путь к изображению (jpg, jpeg, png).
   - `saturation`: Насыщенность цветов (0.5–2.0, по умолчанию 1.0).
   - Ответ: JSON с именем выходного файла (`api_output.jpg`).

3. **Пример в Python**:
   ```python
   import requests

   url = "http://localhost:8000/colorize"
   files = {"file": open("image.jpg", "rb")}
   data = {"saturation": 1.2}
   response = requests.post(url, files=files, data=data)
   print(response.json())
   ```

- Если возникают ошибки, проверьте логи в терминале или обратитесь к [INSTALL.md](INSTALL.md).

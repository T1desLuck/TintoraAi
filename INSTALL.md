# Установка TintoraAI

Хочешь установить TintoraAI на свой компьютер? Не волнуйся, это легко! Следуй этим шагам шаг за шагом, и у тебя всё получится.

## Что тебе понадобится
- Компьютер с **Windows**, **Mac** или **Linux**.
- Установленный **Python 3.8** (скачай с [python.org](https://www.python.org/downloads/release/python-380/)).
- Git (для скачивания кода, установи с [git-scm.com](https://git-scm.com/)).

## Шаги установки

### 1. Убедись, что Python 3.8 установлен
- Открой терминал (на Windows — "Командная строка", на Mac/Linux — "Терминал").
- Введи:
  ```bash
  python --version
  ```
- Если видишь `Python 3.8.x`, всё ок. Если нет, установи Python 3.8 и добавь его в PATH (спроси в Google, как это сделать).

### 2. Установи Git
- Скачай и установи Git с сайта.
- Проверь установку:
  ```bash
  git --version
  ```
- Если работает, переходи дальше.

### 3. Скачай проект
- В терминале введи:
  ```bash
  git clone https://github.com/T1desLuck/TintoraAi.git
  cd TintoraAi
  ```
- Это создаст папку `TintoraAi` с кодом проекта.

### 4. Установи зависимости
- Убедись, что ты в папке `TintoraAi` (проверь командой `dir` на Windows или `ls` на Mac/Linux).
- Установи нужные библиотеки:
  ```bash
  pip install -r requirements.txt
  ```
- Если появляется ошибка, убедись, что `pip` работает (`pip --version`) и обнови его:
  ```bash
  python -m pip install --upgrade pip
  ```

### 5. Проверь установку
- Запусти тесты, чтобы убедиться, что всё работает:
  ```bash
  pytest tests/
  ```
- Если видишь зелёные строки и нет ошибок, установка прошла успешно!

## Если что-то пошло не так
- Проверь, правильно ли ты ввёл команды.
- Убедись, что интернет работает.
- Посмотри ошибки в терминале и спроси в "Issues" на GitHub, если не разберёшься.

Теперь ты готов использовать TintoraAI! Переходи к `USAGE.md`, чтобы начать.

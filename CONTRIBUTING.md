# Как внести вклад в TintoraAI

Спасибо за интерес к проекту `TintoraAI`! Этот документ объясняет, как помочь в разработке.

## Как внести вклад
1. **Форкните репозиторий**:
   ```bash
   git clone https://github.com/<your-username>/TintoraAI.git
   cd TintoraAI
   ```
2. **Создайте ветку**:
   ```bash
   git checkout -b feature/your-feature
   ```
3. **Сделайте изменения**:
   - Следуйте PEP 8 и настройкам `.flake8` (максимальная длина строки 120).
   - Добавьте тесты в `tests/` для новых функций.
4. **Проверьте код**:
   ```bash
   flake8 src/ tests/
   pytest tests/
   ```
5. **Отправьте Pull Request**:
   - Опишите изменения в PR.
   - Убедитесь, что CI проходит (см. `.github/workflows/ci.yml`).

## Идеи для улучшений
- Добавить аугментации данных в `dataset.py`.
- Оптимизировать модель для ещё меньшего потребления памяти.
- Добавить поддержку других форматов изображений.
- Интегрировать новые метрики качества (например, PSNR).

## Создание меток
- Метки `.npy` создаются нейронной сетью [NpyLabelNet](https://github.com/T1desLuck/NpyLabelNet.git).
- Если вы хотите улучшить генерацию меток, внесите изменения в `NpyLabelNet` и обновите датасет.

## Сообщение об ошибках
- Открывайте issue с подробным описанием проблемы.
- Укажите окружение (OS, Python, GPU) и шаги для воспроизведения.

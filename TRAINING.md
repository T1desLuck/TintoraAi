# Обучение TintoraAI

Хочешь обучить TintoraAI на своих чёрно-белых и цветных фото? Это несложно! Здесь всё объяснено по шагам, даже если ты новичок.

## Что нужно для обучения
- Установленный TintoraAI (смотри [INSTALL.md](INSTALL.md)).
- Компьютер с хорошим процессором или видеокартой (GPU ускорит процесс).
- Большое количество фото (минимум 10 тысяч пар ч/б и цветных).

## Шаги обучения

### 1. Подготовь данные
- Создай папку `dataset` рядом с папкой `src`.
- Внутри сделай три подпапки:
  - `bw/` — положи туда чёрно-белые фото (например, `photo1.jpg`).
  - `color/` — положи туда цветные версии тех же фото (например, `photo1.jpg`).
  - `labels/` — положи файлы с метками объектов (например, `photo1.npy` с числами от 0 до 9).
- Убедись, что имена файлов совпадают (например, `photo1.jpg` в `bw/` и `color/`).

### 2. Проверь данные
- Убедись, что все фото в формате `.jpg`, а метки в `.npy`.
- Открой пару файлов, чтобы проверить, что они загружаются (например, в Python с помощью `Image.open()`).

### 3. Запусти обучение
- Открой терминал и перейди в папку проекта:
  ```bash
  cd TintoraAi
  ```
- Запусти скрипт обучения с указанием пути к данным:
  ```bash
  python src/training/train.py --data_path dataset
  ```
- Ты можешь настроить параметры:
  - `--epochs 50` — сколько раз пройти по данным (больше = лучше, но дольше).
  - `--batch_size 8` — сколько фото обрабатывать за раз (меньше, если мало памяти).

### 4. Следи за процессом
- В терминале будешь видеть строки вроде:
  ```
  Epoch 1/50, Loss: 0.85
  ```
- Это значит, модель учится. Чем меньше `Loss`, тем лучше.

### 5. Сохрани модель
- После обучения файл `colorizer_weights.pth` появится в папке проекта.
- Если обучение прервалось, найди файлы вроде `colorizer_weights_epochX.pth` (X — номер эпохи) и используй их.

## Полезные советы
- Если обучение тормозит, уменьши `--batch_size` (например, до 4).
- Для быстрого теста поставь `--epochs 5`, но для хорошего результата используй 50+.
- Если появляется ошибка, проверь данные и спроси в [Issues](https://github.com/T1desLuck/TintoraAi/issues).

## Что делать дальше
- Протестируй модель с помощью `USAGE.md`.
- Если хочешь улучшить, добавь больше фото или настрой параметры в коде.

Удачи в обучении! 🎉
